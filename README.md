# EA Help Support System (Unofficial)

A responsive web-based support forum interface inspired by Electronic Arts Help.  
This project was created as part of a university course assignment to demonstrate front-end development skills using HTML, CSS, and JavaScript.

> ⚠️ Disclaimer: This project is an unofficial student-built application and is **not affiliated with, endorsed by, or associated with Electronic Arts (EA)** in any way.

---

## 📌 Overview

This project simulates a modern support portal similar to EA Help, allowing users to browse platform-specific support sections and forum-style content. The focus of this project is on UI structure, responsive design, and dynamic front-end behavior.

It is intended for educational purposes only.

---

## ✨ Features

- Responsive layout for desktop and mobile devices  
- Platform-based support sections (e.g., PlayStation, Xbox, PC)  
- Forum-style content pages and navigation structure  
- Dynamic content rendering using JavaScript  
- Modular file structure for maintainability  
- Styled UI inspired by modern support portals  

---

## 🛠️ Built With

- HTML5  
- CSS3 (including responsive design techniques)  
- JavaScript (vanilla)
- JSON
